## Дополнение: layout-примитивы Row/Col

- [x] T13a. **shared/ui/stack.tsx (новый)** — `Row`/`Col`: единая точка для
      flex-контейнеров (`gap`/`align`/`justify`/`wrap`, полиморфный `as`)
      вместо ad-hoc `flex items-center gap-*` и `grid gap-*`-как-стек по
      экранам. Мигрированы все подходящие usages (~30) в `app/**`,
      `features/**/ui`, `widgets/**`. Настоящие многоколоночные CSS Grid
      (`sm:grid-cols-2` дат в `tournament-settings-form`, `TabsList
      grid-cols-2` в `auth-dialog`) — намеренно не тронуты: это не
      стек-паттерн, а реальная сетка колонок. `inline-flex` (ссылка «назад» в
      `AdminHeader`) и responsive `hidden … md:flex` (навигация в `navbar`)
      оставлены как точечные исключения через `className`-override.
